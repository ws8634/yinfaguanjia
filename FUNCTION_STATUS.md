## 关键技术选型记录

### 语音识别方案对比

| 方案 | 优点 | 缺点 | 当前状态 | 版本 |
|-----|------|------|---------|------|
| Android原生 | 免费，无需网络 | 识别率低，需弹框 | 备选 | v1.0.0+ |
| 讯飞SparkChain | 识别率高，专业 | 需网络，需授权 | **主方案** | v1.1.0+ |
| 百度语音 | 识别率高 | 需额外集成 | 未使用 | - |

### 数据存储方案对比

| 方案 | 优点 | 缺点 | 当前状态 | 版本 |
|-----|------|------|---------|------|
| 内存存储 | 快速，简单 | 重启丢失 | 已废弃 | v1.0-v1.1 |
| SharedPreferences | 简单，持久 | 不适合大量数据 | **当前使用** | v1.2.0+ |
| Room数据库 | 结构化，查询强 | 复杂，需迁移 | 计划中 | v2.0.0 |
| Firebase | 云端同步 | 需网络，国外服务 | 待评估 | - |

## 配置状态管理

### 当前默认配置 (v1.2.0)

```kotlin
// 语音设置
useRealVoiceApi = true        // 默认使用讯飞语音（非系统语音）

// 天气设置  
useRealWeatherApi = true      // 默认使用和风天气API

// 闹钟设置
reminderEnabled = true        // 默认开启用药提醒
```

## 问题排查指南

### 语音识别问题
- 系统语音黑框弹出 → 检查`useRealVoiceApi`是否为true
- 讯飞语音不识别 → 检查网络连接和API Key
- 识别率低 → 确认使用讯飞SDK而非系统语音

### 闹钟提醒问题
- 不提醒 → 检查电池优化设置、权限设置
- 锁屏不弹出 → 检查锁屏显示权限
- 后台不提醒 → 检查自启动权限

### 数据丢失问题
- 用药记录丢失 → 检查是否使用SharedPreferences存储
- 重启后数据丢失 → 确认不是使用内存存储
